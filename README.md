# Internshala Clone - Full Stack Project

A feature-rich clone of Internshala with advanced security, community features, and multi-language support.

## 6 Core Tasks Implemented

### 1. Public Space (Community Feed)
- **Networking**: Users can connect with other students.
- **Daily Limits**: Posting limits are enforced based on the number of friends (connects) a user has.
- **Discovery**: "People you may know" section for easy networking.

### 2. Forgot Password
- **Secure Generation**: Generates a random 10-character password containing only letters.
- **Rate Limiting**: Users can only reset their password once per day.
- **Integration**: Supports both Email/Phone reset.

### 3. Subscription Payments (Razorpay)
- **Tiered Plans**: Free, Bronze, Silver, and Gold plans.
- **Time Restriction**: Payments are only accepted between **10:00 AM – 11:00 AM IST**.
- **Real-time Activation**: Instant plan upgrade upon successful payment.

### 4. Resume Builder
- **Premium Feature**: Costs ₹50 to generate.
- **Security**: Requires Email OTP verification before initiating payment.
- **Design**: Professional resume generation.

### 5. Multi-Language Support
- **6 Languages**: Supports English, Hindi, Marathi, Gujarati, Tamil, and French.
- **High Security**: Switching to French requires mandatory Email OTP verification.

### 6. Login Tracking & Restrictions
- **Device Tracking**: Logs IP, Browser, OS, and Device type for every login.
- **Chrome Security**: Google Chrome users are required to enter an Email OTP for extra security.
- **Mobile Restriction**: Mobile access is only allowed between **10:00 AM – 1:00 PM IST**.

---

## Technology Stack
- **Frontend**: Next.js, React, Tailwind CSS, Redux Toolkit
- **Backend**: Node.js, Express.js, MongoDB (Mongoose)
- **Auth**: Firebase Authentication (Google Login)
- **Payments**: Razorpay API
- **Emails**: Nodemailer (SMTP)

## Getting Started

### 1. Backend Setup
1. Navigate to `/backend`.
2. Create a `.env` file and add:
   ```env
   DATABASE_URL=
   EMAIL_USER=
   EMAIL_PASS=
   RAZORPAY_KEY_ID=
   RAZORPAY_KEY_SECRET=
   ```
3. Run `npm install`.
4. Start the server: `node index.js`.

### 2. Frontend Setup
1. Navigate to `/internarea`.
2. Create a `.env` file and add:
   ```env
   NEXT_PUBLIC_BACKEND_URL=http://localhost:5000
   ```
3. Run `npm install`.
4. Start the dev server: `npm run dev`.
